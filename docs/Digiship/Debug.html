NAME

Digiship::Debug

Top

SYNOPSIS

use Digiship::Debug;

Top

DESCRIPTION

Digiship::Debug is used in instances where a common debug interface is desired. Rather than creating a debug method in every perl script or module, the same results may be obtained from this module. Additionally, all the debug messages that an application generates will all have the same look and feel.

The ideal way to enable debugging on a global scale within an application is to set a top-level (global) variable called debug that sets the level within this module. In a perfect world, this debug variable would come from a source outside the script itself such as from the command line or from a configuration file.

Top

INSTALLATION

This module is installed by default with the EDIServer application.

Top

METHODS

  • setDebugLevel()


    •      DESCRIPTION : Sets the level of messages to print
         ARGUMENTS : String(level) [0 = off, 1 = on]
         RETURNS : 0 or 1
         NOTES : None

  • getVersion()


    •      DESCRIPTION : Returns the version of the modue
         ARGUMENTS : None
         RETURNS : String(version)
         NOTES : None

  • getName()


    •      DESCRIPTION : Returns the name of the module
         ARGUMENTS : None
         RETURNS : String(Name)
         NOTES : None

  • print()


    •      DESCRIPTION : Prints a properly formatted debug message to STDERR
         ARGUMENTS : String(message)
         RETURNS : 0 or 1
         NOTES : None

    Top

    USAGE

    # Load the module
use Digiship::Debug;

    
# Create a new instance of it
$Debug = Digiship::Debug->new();

    
# Set the debug level to 1
$Debug->setDebugLevel(1);

    
# Print a debug message
$Debug->print("Debug message");

    
# Or like this
$Debug->print("Hello from " . hostname());
